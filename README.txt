Etape d'utilisation
-Récupérer le projet sur github
-Installer rust/OpenSSL/MySQL
-Lancer le script bash pour créer la base de données correspondant au projet: sudo ./script_bdd.sh
-Modifier le chemin dans le fichier de config du certificat intermédiaire
-faire cargo build (cela peut prendre quelques minutes le temps d'installer les dépendences)
-faire cargo run
-Aller sur http://localhost:8080/MenuCrypto.html

Regardez sur le rapport si vous avez besoin de plus de détails
